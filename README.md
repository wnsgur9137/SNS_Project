> # **Git 사용법**

<br>

프로젝트 복제해서 로컬로 다운
```
git clone https://github.com/wnsgur9137/SNS_Project.git
```

<br>

branch 설정
```
git checkout [Branch Name]
```

<br>

파일 commit
```
git add *
git commit -m "commit name"
git push origin [Branch Name]
```

<br>

변경사항 불러오기 (PULL)
```
git pull origin [Branch Name]
```
